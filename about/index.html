arrow_back Covid Tracker Bankura





As we all are aware of the fact that Bankura is getting dozens of COVID-19 cases in recent days. So its our responsibility to have a track on these stuffs. Keeping such requirements in mind Debtanu Dey & Deep Nandi had successfully launched a COVID TRACKER web application which will be particularly working for Bankura District. It is integrated with graphical analysis which are easy to understand. Also age groups are shown from which we get the information that which particular age-group should take more precautions. Block wise data is also reflected over here.
